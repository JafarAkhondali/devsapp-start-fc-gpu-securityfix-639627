## 本地快速体验

通过该应用，您可以简单快速的创建一个 FC 案例到阿里云函数计算服务。

- 下载命令行工具：`npm install -g @serverless-devs/s`
- 初始化一个模版项目：`s init start-fc-http-python3`
- 进入项目后部署项目：`cd start-fc && s deploy`
